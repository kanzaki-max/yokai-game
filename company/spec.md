# 🏮 モックポコ マスター仕様書

## ゲームコンセプト
日常生活で出会う迷惑な行動をする人間（モック）を捕まえ、
優しさで更生させて街を平和にするゆるカジュアルゲーム。

## 世界観・雰囲気
- ゆるくてクスッと笑えるカジュアル系
- 和風テイストを少し混ぜる
- モックへの優しさ・温かさがテーマ

## 画面構成
1. ホーム画面（モック一覧 ＋ 街の平和度メーター）
2. 遭遇画面（目撃通報 → 現場へ）
3. 説明シーン（優しさテキスト → 暗転 → 捕獲へ）
4. 捕獲画面（記憶ゲーム方式）
5. 更生画面（会話・ご飯・遊ぶ・褒める）
6. 殿堂画面（更生した元モック一覧）
7. 街の平和度ページ（解放数で街の様子が変化）
8. デッキ編成画面（ホームからアクセス）

## ゲームループ
遭遇 → 説明シーン → 捕獲（記憶ゲーム） → 更生 → 解放 → 殿堂入り

## 遭遇方法
- 通報システム：「〇〇でモックの目撃情報が入りました！」
- 場所を選んで「タップして確かめる」を押すと必ずモックが出現する
- 出現するモックは生息地に応じてランダムで決まる
  - 電車内：もたれ小僧・降車拒否霊・音漏れ大魔王
  - 会社：武勇伝おじ・会議長老
  - 街中：直進タックル鬼・割り込み忍者・スマホ歩き亡霊・傘ぶんぶん侍
  - コンビニ：割り込み忍者

## 説明シーンの流れ
1. モック名と説明テキストがタイプライター風に表示（優しさ視点）
2. 3秒後にフェードアウト
3. 0.5秒の暗転（BGM停止）
4. 「……。捕まえましょうか。」がポツンと表示
5. 捕獲画面へ切り替わる（bgm-battle.mp3 再生開始）

## 捕獲方法（記憶ゲーム方式）

### 記憶フェーズ
魔法陣の中にランダムな位置で光が順番に点灯→消える。
全て点灯し終わったら「さあ、タップして！」のテキストを表示。

| レアリティ | 光る箇所数 | 点灯時間 | 間隔 |
|-----------|-----------|---------|------|
| N  | 3箇所 | 1.0秒 | 0.5秒 |
| R  | 4箇所 | 0.7秒 | 0.3秒 |
| SR | 5箇所 | 0.5秒 | 0.2秒 |
| UR | 6箇所 | 0.3秒 | 0.15秒 |

### 解答フェーズ
ユーザーが光った順番通りに画面をタップする。
- 正解タップ：金色に輝いてキラッとする演出 ＋ 「ぽん♪」の音
- 不正解タップ：赤くブルッと震える演出 ＋ 「ボッ…」の音

### 捕獲判定
記憶ゲーム終了後、以下の流れで捕獲判定を行う：

1. 正解数に応じた捕獲率で即座に判定を実行
2. 全問正解の場合のみ「パーフェクト！」と小さく一瞬表示してから魔法陣演出へ
3. 不正解ありの場合はそのまま魔法陣演出へ

| 正解数 | 捕獲率 |
|-------|--------|
| 全問正解 | 100%（必ず成功） |
| 1問ミス  | 60% |
| 2問ミス  | 30% |
| 3問以上ミス | 10% |

### 結果演出
- 成功時：魔法陣が全開放してズバッと吸い込まれる → 「捕まえた！」→ ホームへ
- 失敗時：魔法陣が閉じる演出 → 「逃げられた…」→ ホームへ

## タイプシステム（1体1タイプ制）
タイプは笑・無・怒・陽・陰の5種類。

| 技タイプ | 対象タイプ | 倍率 |
|---------|----------|------|
| 笑 → 無 | ×2.0 |
| 無 → 怒 | ×2.0 |
| 怒 → 笑 | ×2.0 |
| 笑 → 怒 / 無 → 笑 / 怒 → 無 | ×0.5 |
| 陽 → 陰 | ×1.5 |
| 陰 → 陽 | ×1.5 |
| 笑・無・怒 vs 陽・陰 | ×1.0 |

## デッキ編成
- ホーム画面の「デッキを編成する」ボタンからアクセス
- 所持スキルから3つ選んで保存（localStorageに保存）
- 未設定時は春風・余白・直言の3つをデフォルト使用
- デッキは捕獲バトルには使用しない（技習得・確認専用）

## 更生方法

モックには5つの状態があり、アクションのたびにランダムで変化する。
状態は画面に直接表示せず、ヒントセリフで示す。

### 状態一覧

| 状態 | ヒントセリフ | 効果的アクション | 逆効果アクション |
|------|------------|----------------|----------------|
| お腹が空いている | 「なんかええ匂いせえへんか…」 | ご飯をあげる (+20%) | それ以外 (-10%) |
| 疲れている | 「ちょっと休ませてくれんか…」 | 褒める・会話する (+20%) | 一緒に遊ぶ (-10%) |
| 一人になりたい | 「…（無言でそっぽを向いている）」 | そっとしておく (+20%) | 会話する・褒める (-10%) |
| 退屈している | 「ヒマじゃのう、なんかないんか」 | 一緒に遊ぶ・会話する (+20%) | ご飯をあげる (-10%) |
| 機嫌が良い | 「まあ、悪い気はせんな」 | 全て (+20%) | なし |

### アクションボタン

- 会話する
- ご飯をあげる
- 一緒に遊ぶ
- 褒める
- そっとしておく（「一人になりたい」状態の時だけ表示）

全てのボタンは同じ見た目に統一。正解・不正解は押した後のセリフでのみわかる。

### リアクションセリフ

- 正解時：「…ありがとな」「ええのう」「悪くないな」などポジティブなセリフ
- 不正解時：「今はそれじゃない」「ほっといてくれ」などのセリフ

更生度は0%以下にはならない。

## 更生完了 → 解放
- 更生度MAXで解放イベント発生
- 元モックが一言コメントを残して旅立つ
- 殿堂入りページに追加される
- 更生完了したらホーム画面へ遷移

## 技習得システム
- 更生完了時に50%の確率で新技を習得
- 習得した技はデッキ編成画面で確認・選択できる

## BGM構成
- bgm-title.mp3 → タイトル画面
- bgm-home.mp3 → ホーム画面
- bgm-encounter.mp3 → 遭遇・探索画面
- bgm-battle.mp3 → 捕獲画面
- bgm-rehab.mp3 → 更生画面
- bgm-farewell.mp3 → 殿堂画面
- 説明シーンは無音（BGM停止）
